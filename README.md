# ProyectoFinal-Martin

Les presento mi proyecto de ecommerce basado en el proyecto del curso de Desarrollo web, avanzando y mejorando con los conceptos de JavaScript.

Entre los conceptos aprendidos, la web le permite al usuario suscribrirse para recibir notificaciones, agregar elementos al carrito seleccionando la talla, eliminando elementos del carrito que no desee y por ultimo, finalizar la compra ingresando los datos de contacto.

Para poder visualizar el listado de productos es necesario correr la web con un Live Server.
